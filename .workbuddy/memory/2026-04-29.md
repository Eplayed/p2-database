## 修复 GitHub Actions 爬虫超时问题

### 问题描述
GitHub Actions 运行爬虫时遇到 `ProtocolError: Protocol error (Page.navigate): Target closed` 错误，导致爬虫失败。

### 修复方案
1. **增加 protocolTimeout**：从 240秒 增加到 600秒（10分钟）
2. **添加页面健康检查**：在每次处理玩家前检查页面是否崩溃，崩溃则自动重建页面
3. **添加页面导航重试逻辑**：`page.goto` 失败时最多重试 3 次，每次失败后等待 5 秒
4. **修复 keystones 提取逻辑**：避免嵌套的 `page.evaluate` 调用（这是导致超时的根本原因）
